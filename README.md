# property-auction

This is a web application written in Clojure language. Application is for collecting information about client who want to participate on property auction. If client exists then is possible to add application for property witch offered. Every client can have only one application for one offering property. When user create new application for property its must first chosen client from list, that list will showed off only client who don’t have 
created application for that property.

## Usage

Download and install Leiningen. http://leiningen-win-installer.djpowell.net/
Download and install MongoDB. http://www.mongodb.org/downloads
Start MongoDB.
Start your application - cd to project location - lein run
